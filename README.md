# FocusFlow

A minimal Pomodoro timer for focused studying.

## Live Demo
https://pomobox.app

## Features
- Focus & Break timer with auto-transition
- Customizable durations (15/25/45/60 min)
- Browser notifications & sound alerts
- Daily session tracking
- Dark theme UI

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- shadcn/ui
